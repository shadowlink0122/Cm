# バージョン計画まとめ (v0.10.0 - v0.14.0)

## v0.10.0 - 安定化リリース ✅ 完了

### 完了機能
- **モジュールシステム**: import/export, 階層的インポート, ワイルドカード
- **FFI**: `use libc { ... }`, std::mem
- **ラムダ式**: `(params) => { body }`
- **クロージャ**: 変数キャプチャ（ClosureValue型）
- **動的スライス**: `T[]`, `T[][]`, 高階関数（map/filter/reduce等）
- **no_std対応**: カスタムアロケータ, 自前メモリ/文字列関数
- **MIR最適化**: 定数畳み込み, DCE, SCCP, CSE, インライン化等
- **Debug/Display**: `with Debug`, `with Display` 自動実装
- **文字列補間拡張**: `{*ptr}` デリファレンス補間
